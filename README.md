# Telecom-Churn-Insight-End-to-End-Predictive-Pipeline-Predictive-Architecture
Built an end-to-end data engineering and machine learning pipeline to identify high-risk telecom subscribers. Ingested CSV data into a MySQL star schema, trained a Random Forest model with 94.5% accuracy, used ELI5 for explainability, and automated database webhooks to support targeted customer retention.
